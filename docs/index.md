# IaC Scanner

Scan **Terraform** and **AWS CDK** Infrastructure-as-Code, get a security and best-practice **report** and **fixed code**. Built with a factory pattern and LangChain (two AIs: one for analysis, one for fixes).

- **Install:** `pip install iac-scanner`
- **PyPI:** [iac-scanner](https://pypi.org/project/iac-scanner/)
- **Source:** [GitHub repository](https://github.com/bishwasjha/iac-scanner)
